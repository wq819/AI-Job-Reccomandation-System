# AI-Based Job Recommendation System
**Prepared by:** Waqaas Hussain | **Subject:** Programming for AI

## 📌 About
AI-powered job recommendation system using TF-IDF Vectorization and Cosine Similarity.
Built with Python + Streamlit. Includes 30 real-world job listings across 10+ categories.

## ✅ Features
- Skill-based job matching using Machine Learning
- TF-IDF + Cosine Similarity engine
- Skill gap analysis (what skills to learn)
- Job market analytics dashboard with charts
- Category filter, salary filter, location filter
- Download results as CSV
- 3-page navigation (Home, Analytics, How It Works)

##  Run Locally
```bash
 -r requirements.txt
streamlit run app.py
```

## ☁️ Deploy on Streamlit Cloud (FREE)
1. Upload all files to a **GitHub repository**
2. Go to **https://share.streamlit.io**
3. Click **New App** → Select your GitHub repo
4. Set main file as **app.py**
5. Click **Deploy** ✅ — Live in 2 minutes!
